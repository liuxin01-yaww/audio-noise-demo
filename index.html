WebRTC 音频通信演示

功能介绍

这是一个基于WebRTC的音频通信演示程序，支持以下功能：

  • 两个Web端之间的实时音频通信
  • 在拉流端录制音频
  • 录制完成后下载音频文件
  • 现代化的用户界面

使用说明

请按照以下步骤使用：

  1. 在两个浏览器窗口（或设备）中分别打开推流端和拉流端
  2. 在推流端点击"开始推流"按钮
  3. 在拉流端点击"开始拉流"按钮
  4. 建立连接后，可以在拉流端录制音频
  5. 录制完成后可以下载音频文件
🎤 推流端 (发送音频) 🎧 拉流端 (接收音频)

技术栈

Node.js + Express + Socket.IO + WebRTC API + MediaRecorder API
